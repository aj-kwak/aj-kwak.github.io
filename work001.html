001 : Web3 FPS Shooter Game

The technology stack used for this project is quite ordinary for an indie developer. Using proven frameworks to piece up the puzzle and relying on C# for custom solutions. Creating a video game used to require a fair amount of technical expertise and resources, but today's technology has made it easier than ever. Though, in a Web3 game; the ecosystem is built on a blockchain, where the in-game assets have an ownership and value that goes beyond the game itself.

Unity Game Engine is a powerful cross-platform 3D engine, based on C#; an object-oriented programming language. Easy enough for the beginners, yet powerful enough for the experts. It is primarily used for real-time 3D projects for various industries across games, animation, automotive, architecture, etc.

Developing a game from scratch provide an in-depth learning curve towards game development. Though, most of the time there are existing frameworks available to speed-up the kickstart of a project. Free and paid assets for Unity projects can be found on Unity Asset Store.

For multiplayer logic, a third party service provider specializing in multiplayer server solutions has been initialized. They offer a relatively painless implementation, including good documentation and a Free Account with 20 CCU (Concurring Users) to test the service for unlimited time. Photon Engine is a global cross-platform multiplayer game backend service for synchronous and asynchronous games and applications.

This game is developed almost completely by using Unity Asset Store -assets, including some licensed external scripts for game modes and additional game functionality. User data and registration are handled with a custom PHP app in SQL database on a Virtual Private Server.

Blockchain Integration

The Web3 Login uses an identifier linked to a Web3 wallet account to enable a blockchain connection. Sending data from the marketplace in to the SQL database and vice versa. To achieve this we use JavaScript and an API. The goal is to send skins (nfts) from the marketplace in to the player's inventory. Items purchased on the marketplace, then, will be usable ingame and visible in the inventory. Function to sending items back to the marketplace in purpose of selling or trading them. Trading against real world assets such as cryptocurrencies, nfts, or fiat currencies.
